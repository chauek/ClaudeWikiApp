## How to Use

Each project has its own folder with an `index.md` file as the main page.
Subpages are `.md` files or subfolders with their own `index.md`.

Connections between nodes are automatically detected by Claude and saved in `_meta/graph.json`.
TODO items from all nodes are aggregated in `_meta/todos.json`.
