## Highlights
- **New Maps view** — any `.html` file in your knowledge base is now surfaced as a visual map, both inline with related notes and in a dedicated sidebar view, rendered in a sandboxed iframe.
- Generate-a-map helper with a ready-to-copy Claude prompt.
- Maps column folds down just like the main sidebar.

## Features
- Render `.html` files under `knowledge/` as HTML maps. They appear as leaves in the department tree next to related `.md` nodes and in a new **Maps** view that lists them globally.
- Map preview runs in a sandboxed iframe (`sandbox="allow-scripts"`) with a header showing the title, file path, and an "Open in browser" button.
- New **+** button in the Maps header opens a help modal with a suggested prompt for generating maps via Claude — Esc or backdrop click to dismiss, one-click copy.
- Collapsible Maps list: chevron toggle folds the left column to 100px; title ellipsis-clips, count and **+** hide, only map titles remain.

## Docs & Chore
- Scaffold `CLAUDE.md` documents the HTML map convention (self-contained, `<title>`, placement next to related nodes); scaffold version bumped to 7.
- Architecture doc updated with the new view, IPC channels (`fs:readHtml`, `fs:listMaps`), and shared types (`TreeItem.type`, `HtmlMap`).
- Require Node.js 24 for development and CI: release workflow upgraded, `.nvmrc` pinned, `engines` + `engine-strict` enforce the version on `npm install`.
