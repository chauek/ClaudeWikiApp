## Highlights
- App now checks for new releases on startup and notifies you in Settings
- NavRail shows a badge when an update is available
- One-click DMG download directly to ~/Downloads

## Features
- Update checker fetches latest GitHub release on startup (3s delay) and on demand from Settings
- Settings view shows update notice with version info and "Download DMG" button
- DMG streams to `~/Downloads/ClaudeWiki-<version>.dmg` with progress indicator, then reveals in Finder
- NavRail badge appears when an update is available
- Polish and English translations for all update UI strings

## Improvements
- CI now publishes unsigned builds as stable releases (not pre-release)
